# Project HARMONIA (NASIS)

**다부처 갈등 조정 — 심리적 수정 내시 바게닝 프로토타입**

행동게임이론의 심리적 편향(손실회피·시기심·자기고양·체면)을 정량화하여  
내시 바게닝 목적함수에 반영하고, 양측 부처의 심리적 저항을 최소화하는  
최적 자원 배분(x_A\*, x_B\*)을 도출합니다.

## 핵심 심리 편향 모델

| 편향 | 기호 | 설명 |
|------|------|------|
| 손실 회피 | λ_i | 준거점 이하 배분 시 과도한 거부감 |
| 시기심 | β_i | 상대가 더 받을 때 느끼는 심리적 손실 |
| 자기고양 | α_i | 자기 몫을 실제보다 과대 인식 |
| 체면 | M_i | 일정 수준 이하 배분은 수용 불가 |
